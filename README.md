# Sentinel AI 🚀

A cybersecurity dashboard that detects and visualizes threats in real-time.

## Features
- Threat detection (SQL, XSS, brute force)
- Real-time logs
- Dashboard analytics
- Interactive globe visualization

## Tech Stack
- React + Vite
- Node.js + Express
- MongoDB
- Three.js

## Run Locally

Frontend:
npm install
npm run dev

Backend:
npm install
npm run dev
